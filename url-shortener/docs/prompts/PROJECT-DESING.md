# Верхнеуровневое описание проекта

1. Это должен быть вебсервис, который написан на Java 21 с использованием Spring Boot, Kafka, Postgresql.
2. Этот сервис должен уметь принимать на вход полную ссылку caйта на вход, а затем возвращать ее укороченную версию.
Маппинг между полной ссылкой и укороченной сохраняется в базу данных.
3. Когда пользователь делает запрос к сервису по сокращенной версии ссылки, то сервис делает редирект по полной ссылке
4. Сервис собирает статистику по кол-ву кликов на сокращенную ссылку
